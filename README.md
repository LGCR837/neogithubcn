# NeoGithubCN

GithubCN 增强版

基于 [GithubCN](https://github.com/JQiue/githubCN) 项目修改

## 使用方法

1. 下载源代码
2. 打开浏览器，进入扩展页，点击“加载已解压的扩展程序”，选择下载的源代码文件夹
3. 在浏览器中打开 Github 页面，即可看到翻译后的内容
